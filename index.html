プログラム復習コンテンツ

pmouseX変数,pmouseY変数

サンプルコードの結果

サンプルコード

　　 void setup(){

　　 　size(480, 120);

　　 　strokeWeight(4);

　　 　stroke(0, 102);

　　 }

　　 void draw(){

　　 　line(mouseX, mouseY, pmouseX, pmouseY);

　　 }

説明文

　pmouseX変数とpmouseY変数は、1つ前のフレームにおけるマウスの位置を記憶して、値に変換する。

　mouseXやmouseYと同様にdraw()が実行されるたびに値が更新される。

　これらの変数を組み合わせてline()関数に組み込むと現在の位置と前回の位置が繋がっている線を描くことができる。
